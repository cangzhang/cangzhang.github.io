[解决]GPG error with apt-get/aptitude/update-manager behind proxy

Do this in Terminal:

$ cd /var/lib/apt
$ sudo mv lists lists.old
$ sudo mkdir -p lists/partial
$ sudo apt-get update

[折腾]回归Ubuntu

重新安装上了Ubuntu……
这次把上网问题解决了。主要是通过虚拟机里的XP上的网……
设置很纠结，要设置两个网卡，一个是桥接，另一个是NAT模式。
在XP里安装CCproxy，设置端口（默认的就好，8080）.查看一下你的本地连接的IP地址。
然后到Ubuntu的系统设置里，设置IP，设置到端口，选择应用到整个系统。
That’s All.
Start your browser now!
Here we go!

And:
这次装了以后进行了一次Update，然后启动时提示”Your screens,graphic cards,input devices cannot be deceted correctly” blabla……
google了一大圈，解决很简单：重新安装你的显卡驱动！

M4A87T主板刷新BIOS

自己的配的电脑，用的是 ASUS M4A87T 主板，华硕最近更新了BIOS程序，估计是这块板子真正要退市的时候到了……

于是也顺便刷新了一下BIOS，更新了一下~

又想自己弄一下硬刷BIOS激活，于是找到了这个帖子

于是重启，Alt+F2进入EZ Flash，进行BIOS刷新。

由于没有先升级BIOS而直接进行刷新修改过的BIOS，我的风扇老是会狂转，机箱嗡嗡作响，这个问题烦了我很长时间了，今天忽然想到会不会是BIOS的问题？于是重新刷新了官方的BIOS，现在问题解决咯~~

因此，对于刷新BIOS，有了以下建议：

先升级到官方升级的BIOS版本，再刷写修改过的BIOS程序进行覆盖，这样可以最大化的减少问题。

如果还是出现问题，那么，终极大法：

刷回官方BIOS！

Update：风扇依然狂转！！！ 头疼ing……
