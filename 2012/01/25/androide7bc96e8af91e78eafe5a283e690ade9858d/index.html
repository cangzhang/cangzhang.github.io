Android编译环境搭配

原文地址：http://blog.iquens.com/files/2709.html
1、安装JDK编译Gingerbread 之后的版本（包括Gingerbread）需要安装JDK6.0

sudo add-apt-repository “deb http://archive.canonical.com/ lucid partner”
sudo apt-get update
sudo apt-get install sun-java6-jdk




2、安装需要的软件包64位系统

sudo apt-get install git-core gnupg flex bison gperf build-essential
zip curl zlib1g-dev libc6-dev lib32ncurses5-dev ia32-libs
x11proto-core-dev libx11-dev lib32readline5-dev lib32z-dev
libgl1-mesa-dev g++-multilib mingw32 tofrodos python-markdown
libxml2-utils




32位系统

sudo apt-get install git-core gnupg flex bison gperf build-essential
zip curl zlib1g-dev libc6-dev libncurses5-dev x11proto-core-dev
libx11-dev libreadline6-dev libgl1-mesa-dev tofrodos python-markdown
libxml2-utils




3.建立符号链接
Ubuntu10.10

sudo ln -s /usr/lib32/mesa/libGL.so.1 /usr/lib32/mesa/libGL.so




Ubuntu11.10

sudo ln -s /usr/lib/i386-linux-gnu/libX11.so.6 /usr/lib/i386-linux-gnu/libX11.so




4.配置USB访问权限新建一个/etc/udev/rules.d/51-android.rules文件，内容如下，把OWNER=”” 里的值改成你的用户名。
`# adb protocol on passion (Nexus One)
SUBSYSTEM==”usb”, ATTR{idVendor}==”18d1”, ATTR{idProduct}==”4e12”, MODE=”0600”, OWNER=”<username>”

fastboot protocol on passion (Nexus One)

SUBSYSTEM==”usb”, ATTR{idVendor}==”0bb4”, ATTR{idProduct}==”0fff”, MODE=”0600”, OWNER=”<username>”

adb protocol on crespo/crespo4g (Nexus S)

adb protocol on maguro (Galaxy Nexus)

SUBSYSTEM==”usb”, ATTR{idVendor}==”04e8”, ATTR{idProduct}==”6860”, MODE=”0600”, OWNER=”<username>”
